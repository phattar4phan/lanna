# Lanna
Thai Small Language Model (ThSLM)

# Dataset
| Category                              |  % | Purpose                                              |
| ------------------------------------- | -: | ---------------------------------------------------- |
| Wikipedia                             | 20 | General knowledge, factual writing                   |
| Books (public domain/licensed)        | 20 | Grammar, long-range context, storytelling            |
| Web articles (high qualitty)           | 15 | Everyday language and diverse topics                 |
| News                                  | 15 | Formal Thai, current vocabulary                      |
| Educational content                   | 10 | Clear explanations and instructional style           |
| Government/public documents           |  5 | Formal writing and official terminology              |
| Programming & technical documentation |  5 | Technical vocabulary and code-related text           |
| Forums / Q&A                          |  5 | Conversational language and question-answer patterns |
| Dialogue        |  5 | Natural chat and assistant-like responses            |
